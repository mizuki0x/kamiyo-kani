# @kamiyo/kani-shim

JavaScript helper for consumers that want to read `kamiyo-kani` CI artifacts.

This package does **not** run formal proofs. It parses proof artifacts generated by the Rust toolchain (`summary.md`, benchmark summaries, SARIF).

## Why this exists

Some teams ship Node-first services and still want verification signals in their release flow. This shim lets them consume Kani outputs while Rust verification runs in CI.

## Example

```ts
import { parseKaniSummary } from "@kamiyo/kani-shim";

const result = parseKaniSummary(`# Kani Summary\n\n- status: PASS\n`);
console.log(result.status);
```
